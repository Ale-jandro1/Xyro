# Xyro
Nuestra app va a ser una página web para personas que salen del colegio y no saben que van a estudiar la página va a tener cuestionarios y informacion sobre las carreras va a tener cuestionarios para saber cuáles son tus gustos y así poder ayudarte a escoger qué camino tomar
Test vocacional → Preguntas para descubrir intereses y habilidades del estudiante.
FUNCIONALIDADES
Recomendación de carreras → Mostrar carreras que coincidan con los resultados del test.
Información de carreras → Explicar qué se estudia, duración y campo laboral.
Universidades disponibles → Mostrar universidades donde se puede estudiar cada carrera.
Guardar favoritos → Permitir guardar carreras para revisarlas después.

Herramientas
Android Studio → Para desarrollar la aplicación.
Firebase → Base de datos, usuarios y almacenamiento.
MySQL → Base de datos para guardar información.
Visual Studio Code → Editor de código para programar.
Figma → Diseño de la interfaz y prototipos.
Visión: Convertir a Xyro en una aplicación líder que ayude a estudiantes a descubrir sus habilidades, intereses y metas, brindándoles orientación para elegir una carrera y construir un mejor futuro académico y profesional.
